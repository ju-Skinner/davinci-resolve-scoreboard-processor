# 🏐 DaVinci Resolve Volleyball Scoreboard Automation

Automated volleyball scoreboard generation for **DaVinci Resolve / Fusion** using the Python scripting API.

This project programmatically drives a Fusion-based scoreboard by writing **stepped keyframes** to custom controls, allowing match data to automatically populate scores, sets, and timeouts directly on the timeline.
